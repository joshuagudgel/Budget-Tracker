# Budget Tracker

Exploring a NoSQL solution with MongoDB while organizing my finances.

## Backend

### Stack

Node.js, Express, MongoDB

### Fuctionality

Crud operations for Expenses and Categories.

# Frontend

### Stack

React, TypeScript

### Functionality

Displays all expenses.

## Developers

### Setup

- Docker Desktop recommended
- Clone repo

### Docker commands

Interact with docker container from project root.

#### Start application

```bash
docker-compose up --build
```

#### Stop application

```bash
docker-compose down
```

#### Stop application and remove volumes

```bash
docker-compose down -v
```

#### Access Points

- **Frontend**: http://localhost:3001
- **Backend API**: http://localhost:3000/api
- **MongoDB**: localhost:27017
